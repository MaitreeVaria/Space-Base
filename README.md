# SpaceBase

Welcome to SpaceBase! 🚀

SpaceBase is your one-stop destination for comprehensive space-related information, timelines, and up-to-date news on the cosmos. Explore the mysteries of the universe, track the International Space Station (ISS) in real-time, view stunning astronomy pictures, and stay informed with the latest space news.

**Live Website:** [SpaceBase](https://strange-gold-seagull.cyclic.app/)

## Technologies Used

SpaceBase was built using the following technologies:

- **Node.js**: For server-side scripting.
- **HTML**: For creating the website's structure.
- **CSS**: For styling and design.
- **JavaScript**: For dynamic functionality.

## APIs Used

SpaceBase relies on several external APIs to provide you with the best space-related experience:

- **Timeline**: We use [Knight Lab Timeline JS](https://timeline.knightlab.com/) to present space-related events and timelines in an interactive and engaging way.

- **ISS Location Information**: Keep track of the International Space Station's location in real-time with data from [Where the ISS at?](https://wheretheiss.at/).

- **World Map for ISS Live Location**: Visualize the ISS's path across the globe using [MapQuest](https://www.mapquest.com/).

- **Astronomy Picture of the Day**: Enjoy a daily dose of awe-inspiring cosmic visuals from [NASA's Astronomy Picture of the Day](https://apod.nasa.gov/apod/astropix.html).

- **Live News**: Stay up-to-date with the latest space news courtesy of [Spaceflight News API](https://www.spaceflightnewsapi.net/).
